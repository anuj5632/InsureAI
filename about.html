About InsureAI

InsureAI is a predictive analytics web application built using the power of machine learning. It aims to estimate medical insurance costs for individuals based on key personal and lifestyle factors such as age, BMI, smoking status, and geographic region. By collecting this information and applying trained models, the platform generates personalized cost predictions with a high degree of accuracy.

How It Works

  • Users input basic data like age, sex, BMI, number of children, smoking habits, and location.
  • A machine learning model (trained on real-world insurance datasets) processes this input.
  • The model then predicts the most likely insurance premium range for that profile.
  • Results are presented instantly with a user-friendly interface and visuals.
